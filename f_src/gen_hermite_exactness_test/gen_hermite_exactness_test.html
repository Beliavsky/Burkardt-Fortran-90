gen_hermite_exactness_test

gen_hermite_exactness_test, a FORTRAN90 program which calls gen_hermite_exactness(), which investigates the polynomial exactness of a generalized Gauss-Hermite quadrature rule for the infinite interval (-oo,+oo).
