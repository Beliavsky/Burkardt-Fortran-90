fd1d_advection_diffusion_steady_test

fd1d_advection_diffusion_steady_test, a FORTRAN90 code which calls fd1d_advection_diffusion_steady(), which applies the finite difference method to solve the steady advection diffusion equation v*ux-k*uxx=0 in one spatial dimension, with constant velocity v and diffusivity k, creating graphics files for processing by GNUPLOT.
