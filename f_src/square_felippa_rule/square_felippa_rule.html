SQUARE_FELIPPA_RULE
Quadrature Rules for a Square in 2D

SQUARE_FELIPPA_RULE, a FORTRAN90 code which generates the points and weights of a Felippa quadrature rule over the interior of a square in 2D.

Actually, the word "square" is meant to designate any quadrature region defined by: 

        A(1) <= X <= B(1)
        A(2) <= Y <= B(2)
