sphere_voronoi_test

sphere_voronoi_test, a FORTRAN90 code which calls sphere_voronoi(), which computes and plots the Voronoi diagram of points on the unit sphere.

Related Data and Programs:

sphere_voronoi, a FORTRAN90 code which computes and plots the Voronoi diagram of points on the surface of the unit sphere in 3D.

Source Code:

  • sphere_voronoi_test.sh, runs all the tests.
  • sphere_voronoi_test.txt, the output file.

GEN_00010 is a test set of 10 points:

  • gen_00010.xyz, the point coordinates.
  • gen_00010.png, a PNG image of the Voronoi diagram for the data.
  • gen_00010.voronoi.xyz, the coordinates of the Voronoi vertices.
  • gen_00010.voronoi.xyzf, the indices of the Voronoi vertices that form each face.
