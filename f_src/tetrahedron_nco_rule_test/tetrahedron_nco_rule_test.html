tetrahedron_nco_rule_test

tetrahedron_nco_rule_test, a FORTRAN90 code which calls tetrahedron_nco_rule(), which defines the weights and abscisass for a sequence of 7 Newton-Cotes open quadrature rules over the interior of a tetrahedron in 3D.
