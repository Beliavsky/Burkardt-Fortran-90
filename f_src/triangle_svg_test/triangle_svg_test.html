triangle_svg_test

triangle_svg_test, a FORTRAN90 code which calls triangle_svg(), which uses Scalable Vector Graphics (SVG) to create a plot of a triangle and any number of points. Such plots can illustrate quadrature rules or sampling methods for the triangle.
