gegenbauer_exactness_test

gegenbauer_exactness_test, a FORTRAN90 program which calls gegenbauer_exactness(), which investigates the polynomial exactness of a Gauss-Gegenbauer quadrature rule for the interval [-1,1] with a weight function.

Related:

gegenbauer_exactness, a FORTRAN90 code which tests the monomial exactness of Gauss-Gegenbauer quadrature rules.

Source Code:

  • gegenbauer_exactness_test.sh, runs all the tests.
  • gegenbauer_exactness_test.txt, the output file.

GEGEN_O8_A0.5 is a Gauss-Gegenbauer order 8 rule with ALPHA = 0.5.

  • gegen_o8_a0.5_x.txt, the abscissas of the rule.
  • gegen_o8_a0.5_w.txt, the weights of the rule.
  • gegen_o8_a0.5_r.txt, defines the region for the rule.
