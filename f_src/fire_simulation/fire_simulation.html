fire_simulation

fire_simulation, a FORTRAN90 code which simulates a forest fire over a rectangular array of trees, starting at a single random location.

Languages:

fire_simulation is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
