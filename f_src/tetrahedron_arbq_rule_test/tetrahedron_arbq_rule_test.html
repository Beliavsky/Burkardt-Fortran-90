tetrahedron_arbq_rule_test

tetrahedron_arbq_rule_test, a FORTRAN90 code which calls tetrahedron_arbq_rule(), which returns quadrature rules, with exactness up to total degree 15, over the interior of a tetrahedron in 3D, by Hong Xiao and Zydrunas Gimbutas.
