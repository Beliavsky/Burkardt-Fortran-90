lagrange_interp_2d_test

lagrange_interp_2d_test, a FORTRAN90 code which calls lagrange_interp_2d(), which defines and evaluates the Lagrange polynomial p(x,y) which interpolates a set of data depending on a 2D argument that was evaluated on a product grid, so that p(x(i),y(j)) = z(i,j).
