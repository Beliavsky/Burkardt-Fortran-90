CHEBYSHEV2_RULE
Gauss-Chebyshev Type 2 Quadrature Rules

CHEBYSHEV2_RULE, a FORTRAN90 code which generates a specific Gauss-Chebyshev type 2 quadrature rule, based on user input.

The rule is written to three files for easy use as input to other codes.

The Gauss-Chevbyshev type 2 quadrature rule is used as follows: 

        Integral ( A <= x <= B ) f(x)  sqrt ( ( x - A ) * ( B - x ) ) dx
is to be approximated by 
        Sum ( 1 <= i <= order ) w(i) * f(x(i))

Usage:

chebyshev2_rule order a b filename
where
  • order is the number of points in the quadrature rule.
  • a is the left endpoint.
  • b is the right endpoint.
  • filename specifies the output filenames. filename_w.txt, filename_x.txt, and filename_r.txt, containing the weights, abscissas, and interval limits.
