fd1d_heat_explicit_test

fd1d_heat_explicit_test, a FORTRAN90 code which calls fd1d_heat_explicit(), which solves the time-dependent 1D heat equation, using the finite difference method in space, and an explicit version of the method of lines to handle integration in time.

Related Data and codes:

fd1d_heat_explicit, a FORTRAN90 code which uses the finite difference method (FDM) and explicit time stepping to solve the time dependent heat equation in 1D.
