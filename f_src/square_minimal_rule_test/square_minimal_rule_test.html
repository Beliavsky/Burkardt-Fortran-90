square_minimal_rule_test

square_minimal_rule_test, a FORTRAN90 code which calls square_minimal_rule(), which returns "almost minimal" quadrature rules, with exactness up to total degree 55, over the interior of the symmetric unit square in 2D, by Mattia Festa and Alvise Sommariva.
