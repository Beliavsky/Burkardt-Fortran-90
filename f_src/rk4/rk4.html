rk4

rk4 a FORTRAN90 code which implements a simple Runge-Kutta solver for an initial value problem.

Languages:

rk4 is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

midpoint, a FORTRAN90 code which solves one or more ordinary differential equations (ODEs) using the midpoint method.

NMS, a FORTRAN90 code which includes the DDRIV package of ODE solvers.

ode, a FORTRAN90 code which solves a system of ordinary differential equations, by Shampine and Gordon.

rk4_test

RKF45, a FORTRAN90 code which implements the Runge-Kutta-Fehlberg ODE solver.

TEST_ODE, a FORTRAN90 code which contains routines which define some test problems for ODE solvers.
