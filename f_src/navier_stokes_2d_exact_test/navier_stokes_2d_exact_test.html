navier_stokes_2d_exact_test

navier_stokes_2d_exact_test, a FORTRAN90 code which calls navier_stokes_2d_exact(), which evaluates exact solutions to the incompressible time-dependent Navier-Stokes equations (NSE) over an arbitrary domain in 2D.
