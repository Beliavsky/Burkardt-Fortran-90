fem2d_poisson_rectangle_linear_test

fem2d_poisson_rectangle_linear_test, a FORTRAN90 code which calls fem2d_poisson_rectangle_linear(), which solves the 2D Poisson equation using the finite element method over a rectangular region using a uniform mesh of linear triangular elements.

fem2d_poisson_rectangle_linear, a FORTRAN90 code which solves the 2D Poisson equation on a rectangle, using the finite element method (FEM), and piecewise linear triangular elements.
