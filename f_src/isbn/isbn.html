ISBN
Check Digit for International Standard Book Number

ISBN, a FORTRAN90 code which computes the check digit for an International Standard Book Number (ISBN), or reports whether a given ISBN is valid.

Languages:

ISBN is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

isbn_test

LUHN, a FORTRAN90 code which computes the Luhn check digit for a string, and can validate a string.

UPC, a FORTRAN90 code which determines the check digit for a Uniform Product Code (UPC) or reports whether a given UPC is valid.

Source Code:

  • isbn.f90, the source code.
