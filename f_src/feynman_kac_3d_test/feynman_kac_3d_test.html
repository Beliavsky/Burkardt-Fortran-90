feynman_kac_3d_test

feynman_kac_3d_test, a FORTRAN90 code which calls feynman_kac_3d(), which demonstrates the use of the Feynman-Kac algorithm to solve Poisson's equation in a 3D ellipsoid by averaging stochastic paths to the boundary.
