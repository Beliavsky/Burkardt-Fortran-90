FFTPACK4
Fast Fourier Transform Package

FFTPACK4, a FORTRAN90 code which computes Fast Fourier Transforms (FFTs).

Special features include:

  • real or complex data can be handled;
  • separate routines for forward analysis (data => Fourier coefficients) and backward analysis (Fourier coefficients => data);
  • special "easy" routines;
  • sine and cosine transform routines;
  • quarter wave sine and cosine transform routines;
  • the amount of data is NOT required to be a power of 2.

In most cases, you will need to call several routines in order to get your results. In the simplest case, you will need to call just two routines:

  • an initialization routine;
  • the corresponding forward or backward computational routine.

In more complicated situations, you might want to call three routines:

  • an initialization routine;
  • the corresponding forward or backward computational routine;
  • the corresponding backward or forward computational routine.
In a few cases, a single routine may be used for both the forward and backward computation.

Sometimes, a routine has been so heavily optimized that it is impossible to determine if it is really a correct implementation of the relatively simple Fourier formulas. In some cases, a "slow" version of a transform routine has been supplied, simply to provide a simple check that the formulas are correct.

Licensing:

Most of the code presented here is a translation into FORTRAN90 of the corresponding code in version 4.0 of the FORTRAN77 FFTPACK code. According to the NCAR web site, version 4.0 of FFTPACK is in the public domain.

Routines added to the code by John Burkardt are distributed under the GNU LGPL license.

Languages:

FFTPACK4 is available in a FORTRAN90 version

Related Data and codes:

fft_serial, a FORTRAN90 code which demonstrates the computation of a Fast Fourier Transform, and is intended as a starting point for implementing an OpenMP parallel version.

FFT_OPENMP, a FORTRAN90 code which demonstrates the computation of a Fast Fourier Transform in parallel, using OpenMP.

fftpack4_test

FFTPACK5, a FORTRAN90 code which implements the Fast Fourier Transform.

FFTPACK51, a FORTRAN90 code which implements the Fast Fourier Transform.

fftw_test, FORTRAN90 codes which illustrate the use of the FFTW3 code for Fast Fourier Transforms.

NMS a FORTRAN90 code which includes a wide variety of numerical software, including a copy of some of FFTPACK4.

SFTPACK a FORTRAN90 code which implements the "slow" Fourier transform, intended as a teaching tool and comparison with the fast Fourier transform.

SLATEC a FORTRAN90 code which collects together a number of standard numerical libraries, including a copy of some of FFTPACK4.

Author:

The code presented here is based on version 4.0 of FFTPACK, written in FORTRAN77, by Paul Swarztrauber and Richard Valent, of the National Center for Atmospheric Research (NCAR).

Additions and modifications to the code, and conversion to FORTRAN90, were made by John Burkardt.
