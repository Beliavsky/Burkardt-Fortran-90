legendre_rule_fast_test

legendre_rule_fast_test, a FORTRAN90 code which calls legendre_rule_fast(), which implements a fast algorithm for the computation of the points and weights of the Gauss-Legendre quadrature rule.

Related Data and Programs:

legendre_rule_fast, a FORTRAN90 code which uses a fast algorithm to compute a Gauss-Legendre quadrature rule for estimating the integral of a function with density rho(x)=1 over the interval [-1,+1].
