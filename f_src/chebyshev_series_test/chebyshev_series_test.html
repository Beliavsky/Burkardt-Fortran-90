chebyshev_series_test

chebyshev_series_test, a FORTRAN90 code which calls chebyshev_series(), which can evaluate a Chebyshev series approximating a function f(x), while efficiently computing one, two or three derivatives of the series, which approximate f'(x), f''(x), and f'''(x), by Manfred Zimmer.
