line_cvt_lloyd_test

line_cvt_lloyd_test, a FORTRAN90 code which calls line_cvt_lloyd(), which carries out Lloyd's iteration for computing a Centroidal Voronoi Tesselation (CVT) of points over the interior of a line segment in 1D.

Related Data and Programs:

line_cvt_lloyd, a FORTRAN90 code which applies Lloyd's iteration repeatedly to a set of N points, to compute a Centroidal Voronoi Tessellation (CVT) over the interior of a line segment in 1D.
