JACOBI_EXACTNESS
Exactness of Gauss-Jacobi Quadrature Rules

JACOBI_EXACTNESS, a FORTRAN90 code which investigates the polynomial exactness of a Gauss-Jacobi quadrature rule for the interval [-1,1] with a Jacobi weight function.

Related Data and Programs:

jacobi_exactness, a FORTRAN90 code which tests the monomial exactness of quadrature rules for integrals that include a Jacobi weight.
