tet_mesh_q2l_test

tet_mesh_q2l_test, a FORTRAN90 code which calls tet_mesh_q2l(), which reads information describing a tet mesh of a set of points using 10-node ("quadratic") triangles, and creates a 4-node ("linear") tet mesh.
