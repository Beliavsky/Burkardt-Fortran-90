tet_mesh_to_gmsh_test

tet_mesh_to_gmsh_test, a FORTRAN90 code which calls tet_mesh_to_gmsh(), which converts a pair of files defining a tetrahedral mesh into an equivalent file using the gmsh() msh format.

Related Programs:

tet_mesh_to_gmsh, a FORTRAN90 code which writes a tetrahedral mesh to a gmsh() file.

Source Code:

  • tet_mesh_to_gmsh_test.sh, runs all the tests.

  • mesh_nodes.txt, the nodes for the 4-node tet mesh.
  • mesh_elements.txt, the elements for the 4-node tet mesh.
  • mesh.msh, the Gmsh msh file version of the data.
