sphere_delaunay_test

sphere_delaunay_test, a FORTRAN90 code which calls sphere_delaunay(), which computes the Delaunay triangulation of points on the unit sphere.

Related Data and Programs:

sphere_delaunay, a FORTRAN90 code which computes and plots the Delaunay triangulation of points on the surface of the unit sphere in 3D.

Source Code:

  • sphere_delaunay_test.sh, runs all the tests.
  • sphere_delaunay_test.txt, the output file.

GEN_00010 is a test set of 10 points:

  • gen_00010.xyz, the point coordinates.
  • gen_00010.png, a PNG image of the Delaunay triangulation.
  • gen_00010.xyzl, the XYZL file listing the indices of points to connect to form the Delaunay triangulation.
