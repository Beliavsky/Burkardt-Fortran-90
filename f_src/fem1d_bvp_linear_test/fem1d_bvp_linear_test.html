fem1d_bvp_linear_test

fem1d_bvp_linear_test, a FORTRAN90 code which calls fem1d_bvp_linear(), which applies the finite element method, with piecewise linear elements, to a two point boundary value problem in one spatial dimension, and compares the computed and exact solutions with the L2 and seminorm errors.

fem1d_bvp_linear, a FORTRAN90 code which applies the finite element method (FEM), with piecewise linear elements, to a two point boundary value problem (BVP) in one spatial dimension, and compares the computed and exact solutions with the L2 and seminorm errors.
