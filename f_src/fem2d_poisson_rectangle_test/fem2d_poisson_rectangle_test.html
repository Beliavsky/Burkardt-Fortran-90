fem2d_poisson_rectangle_test

fem2d_poisson_rectangle_test, a FORTRAN90 code which calls fem2d_poisson_rectangle(), which solves the 2D Poisson equation using the finite element method, over a rectangular domain with a uniform mesh of triangular elements.

Related Data and codes:

fem2d_poisson_rectangle, a FORTRAN90 code which solves the 2D Poisson equation on a rectangle, using the finite element method (FEM), and piecewise quadratic triangular elements.

Source Code:

  • fem2d_poisson_rectangle_test.sh, runs all the tests.
  • fem2d_poisson_rectangle_test.txt, the output file.

  • rectangle_nodes.png, a PNG image of the nodes, for NX = NY = 7 (the picture is hard to read for larger values of NX and NY);
  • rectangle_nodes.txt, a text file containing a list, for each node, of its X and Y coordinates;
  • rectangle_elements.png, a PNG image of the elements, for NX = NY = 7 (the picture is hard to read for larger values of NX and NY);
  • rectangle_elements.txt, a text file containing a list, for each element, of the six nodes that compose it;
  • rectangle_solution.txt, a text file containing the solution U at each node (X,Y);
