tester

tester, a BASH code which calls the test programs for FORTRAN90 codes.

For a FORTRAN90 code called "prog.f90", stored in the "prog" directory, a test package may be provided. If so, it will be in the "prog_test" directory, with a source code named "prog_test.f90" and a script to run the test called "prog_test.sh". The tester script will find and run all the test codes that satisfy a given pattern in an ls command, that should be set by the user.
