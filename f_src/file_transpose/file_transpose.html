FILE_TRANSPOSE
"Transpose" a File

FILE_TRANSPOSE, a FORTRAN90 code which creates a "transposed" copy of a file, in which, essentially, the I-th word of the J-th line becomes the J-th word of the I-th line.

Usage:

file_transpose input output
where
  • input is the name of the input file,
  • output is the name of the output file to be created.

For example, if the input file is: 

        Apple Ball    Carrot Doll
         Egg    Fig  1.23456,  Haystack
        Iodine 
               Joystick Kahlua Lompoc     Millipede Negatory
the transposed file will be: 
        Apple Egg Iodine Joystick
        Ball Fig Kahlua
        Carrot 1.23456, Lompoc
        Doll Haystack Millipede
        Negatory
