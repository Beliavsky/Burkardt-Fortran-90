tetrahedron01_monte_carlo_test

tetrahedron01_monte_carlo_test, a FORTRAN90 code which calls tetrahedron01_monte_carlo(), which uses the Monte Carlo method to estimate the integral of a function F(X,Y,Z) over the interior of the unit tetrahedron in 3D.
