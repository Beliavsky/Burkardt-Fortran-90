fd1d_predator_prey_test

fd1d_predator_prey_test, a FORTRAN90 code which calls fd1d_predator_prey(), which solves a predator-prey system in a one dimensional region, creating graphics files for processing by GNUPLOT.

Related Data and codes:

fd1d_predator_prey, a FORTRAN90 code which implements the finite difference method (FDM) for a predator-prey system with time and 1D spatial dependence, creating graphics files for processing by gnuplot, by Marcus Garvie.

Source Code:

  • fd1d_predator_prey_test.f90, typical user routines to initialize U and V;
  • fd1d_predator_prey_test.sh, runs all the tests.
  • fd1d_predator_prey_test.txt, the output file.

  • input.txt, interactive input from the user.

  • uv_data.txt, a graphics data file of (X,U(X),V(X)) values at the final time.
  • uv_commands.txt, a graphics command file to plot the data with GNUPLOT.
  • uv.png, a PNG image of the U and V data at the final time.
