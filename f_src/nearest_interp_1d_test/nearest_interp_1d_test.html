nearest_interp_1d_test

nearest_interp_1d_test, a FORTRAN90 code which calls nearest_interp_1d(), which interpolates a set of data using a piecewise constant interpolant defined by the nearest neighbor criterion, creating graphics files for processing by GNUPLOT.
