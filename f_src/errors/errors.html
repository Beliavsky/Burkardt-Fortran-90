ERRORS
Failures of Numerical Algorithms

ERRORS, a FORTRAN90 program which demonstrates how reasonable computations can produce numerical nonsense. This illustrates that the programmer must not assume that a numerical algorithm that seems reasonable will always produce correct and reliable results.

The computations include polynomial evaluation and root finding, linear system solution, minimization, and Taylor series approximation.
