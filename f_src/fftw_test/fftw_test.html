fftw_test

fftw_test, FORTRAN90 codes which demonstrate the use of fftw, a fast Fourier transform code, which was written by Matteo Frigo and Steven Johnson.

FFTW is a code of C routines which can compute the Fast Fourier transform very efficiently.

FFTW can compute the discrete Fourier transform of:

  • complex data;
  • real data;
  • even-symmetric real data (cosine transform);
  • odd-symmetric real data (sine transform);
FFTW can also compute the discrete Hartley transform of real data.

The length of the data is not required to be a power of 2. Any length of data can be processed.

The data is not restricted to 1 dimensional vectors. FFTW can handle data that is 2 or 3 dimensional. In fact, the code can handle data with any dimensionality.

Interested users should go to the FFTW home page: http://www.fftw.org/ which contains the current source code, examples, and documentation.

Usage:

Any code, subroutine or function that calls the FFTW functions must include, in its declarations, the statement 


        include "fftw3.f90"
which will add the appropriate declarations for the FFTW functions.

At link time, you must include the compiled FFTW code, which is probably a file with the name libfftw.a. If directory is the directory where this code is stored, then an appropriate command might be 


        gfortran myprog.f90 -Ldirectory -lfftw
