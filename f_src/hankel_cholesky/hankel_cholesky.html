HANKEL_CHOLESKY
Cholesky Factor of a Positive Definite Hankel Matrix

HANKEL_CHOLESKY, a FORTRAN90 library which can compute the upper Cholesky factor of a positive definite symmetric (PDS) Hankel matrix H, that is, H = R' * R.

A Hankel matrix is a matrix which is constant along all antidiagonals. A schematic of a 5x5 symmetric Hankel matrix would be: 

        a  b  c  d  e
        b  c  d  e  f
        c  d  e  f  g
        d  e  f  g  h
        e  f  g  h  i

hankel_cholesky_test

HANKEL_PDS, a FORTRAN90 library which can compute a lower triangular matrix L which is the Cholesky factor of a positive definite (symmetric) Hankel matrix H, that is, H = L * L'.

TOEPLITZ_CHOLESKY, a FORTRAN90 library which computes the Cholesky factorization of a nonnegative definite symmetric Toeplitz matrix.
