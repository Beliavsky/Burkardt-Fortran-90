QUAD_MPI
Quadrature using MPI

QUAD_MPI, a FORTRAN90 code which approximates an integral using a quadrature rule. The computation is done in parallel by using MPI.

Languages:

QUAD_MPI is available in a C version and a C++ version and a FORTRAN90 version and a Python version.
