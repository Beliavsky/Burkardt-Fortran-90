sparse_display_test

sparse_display_test, a FORTRAN90 code which calls sparse_display(), which can read information defining a matrix of numbers and display the sparsity pattern or location of the nonzero elements using gnuplot. This operation is already available in the built-in MATLAB "spy" command.
