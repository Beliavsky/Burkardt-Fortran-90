three_body_ode_test

three_body_ode_test, a FORTRAN90 code which calls three_body_ode(), which simulates the solution of the planar three body problem.

Related Data and Programs:

three_body_ode, a FORTRAN90 code which defines a set of ordinary differential equations (ODE) which simulates the behavior of three planets, constrained to lie in a plane, and moving under the influence of gravity, by Walter Gander and Jiri Hrebicek.

Source Code:

  • three_body_ode_test.f90, calls all the tests.
  • three_body_ode_test.sh, runs all the tests.
  • three_body_ode_test.txt, the output file.

  • three_body_ode_t.txt, the values of time T at every 0.1 seconds.
  • three_body_ode_y.txt, the values of the 12 state variables at every time.
