LOG_NORMAL_TRUNCATED_AB
The Log Normal PDF Truncated to [A,B]

LOG_NORMAL_TRUNCATED_AB, a FORTRAN90 code which can evaluate quantities associated with the log normal Probability Density Function (PDF) truncated to the interval [A,B].
