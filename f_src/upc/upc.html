UPC
Uniform Product Code Check Digit

UPC, a FORTRAN90 code which can compute the check digit associated with a uniform product code (UPC), or it can report whether a 12-digit UPC is actually valid.

Related Data and Programs:

ISBN, a FORTRAN90 code which can determine the check digit for an International Standard Book Number or report whether a given ISBN is valid.

LUHN, a FORTRAN90 code which computes the Luhn check digit for a string, and can validate a string.
