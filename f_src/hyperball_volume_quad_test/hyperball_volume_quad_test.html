hyperball_volume_quad_test

hyperball_volume_quad_test, a FORTRAN90 program which calls hyperball_volume_quad(), which investigates the behavior of a quadrature rule when it is applied to the characteristic function of the unit M-dimensional hyperball, which is a discontinuous function.
