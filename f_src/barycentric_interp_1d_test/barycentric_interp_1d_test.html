barycentric_interp_1d_test

barycentric_interp_1d_test, a FORTRAN90 code which calls barycentric_interp_1d(), which defines and evaluates the Lagrange polynomial p(x) which interpolates a set of data, so that p(x(i)) = y(i). Because a barycentric formulation is used, polynomials of very high degree can safely be used.
