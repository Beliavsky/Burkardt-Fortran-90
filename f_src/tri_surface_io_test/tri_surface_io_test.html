tri_surface_io_test

tri_surface_io_test, a FORTRAN90 code which calls tri_surface_io(), which reads or writes the 3D graphics information stored in a pair of files that represent a TRI_SURFACE, that is, an object whose surface is represented by a collection of triangles.
