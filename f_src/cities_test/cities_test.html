cities_test

cities_test, a FORTRAN90 code which calls cities(), which works with problems involving intercity distances.

cities, a FORTRAN90 code which handles various problems associated with a set of cities on a map.

  • uscap_ll.txt, the latitude and longitude of each capital.
  • uscap_xy.txt, the (X,Y) coordinates of each capital using a cylindrical projection.
  • wg22_dist.txt, a distance table.
  • wg22_xy.txt, the (X,Y) coordinates of each city.
