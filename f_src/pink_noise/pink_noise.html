PINK_NOISE
Samples of a Pink Noise Signal

PINK_NOISE, a FORTRAN90 code which can generate random values taken from an approximate pink noise signal obeying a 1/f power law.

Languages:

PINK_NOISE is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.
