square_symq_rule_test

square_symq_rule_test, a FORTRAN90 code which calls square_symq_rule(), which returns symmetric quadrature rules, with exactness up to total degree 20, over the interior of the symmetric square in 2D, by Hong Xiao and Zydrunas Gimbutas.
