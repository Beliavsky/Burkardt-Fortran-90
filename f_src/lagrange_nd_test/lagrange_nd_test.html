lagrange_nd_test

lagrange_nd_test, a FORTRAN90 code which calls lagrange_nd(), which is given a set of ND points X(*) in D-dimensional space, and constructs a family of ND Lagrange polynomials P(*)(X), associating polynomial P(i) with point X(i), such that, for 1 <= i <= ND, 

        P(i)(X(i)) = 1
but, if i =/= j 
        P(i)(X(j)) = 0
