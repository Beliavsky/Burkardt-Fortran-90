WEDGE_GRID
Grid Points Within the Unit Wedge

WEDGE_GRID, a FORTRAN90 code which generates a grid of points over the interior of the unit wedge in 3D.

The interior of the unit wedge in 3D is defined by the constraints: 

        0 <= X
        0 <= Y
             X + Y <= 1
       -1 <= Z <= +1
