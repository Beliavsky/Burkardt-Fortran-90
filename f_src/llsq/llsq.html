LLSQ
Linear Least Squares Problem for Y = A*X+B

LLSQ is a FORTRAN90 code which solves the simple linear least squares (LLS) problem of finding the formula of a straight line y=a*x or y=a*x+b which minimizes the root-mean-square error to a set of N data points.
