tetrahedron_properties_test

tetrahedron_properties_test, a FORTRAN90 code which calls tetrahedron_properties(), which computes properties including the centroid, circumsphere, dihedral angles, edge lengths, face angles, face areas, insphere, quality, solid angles, and volume of a tetrahedron in 3D.
