DUTCH
Computational Geometry

DUTCH, a FORTRAN90 code which implements some of the computational geometry routines from the reference.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

DUTCH is available in a FORTRAN90 version.

Related Data and Programs:

dutch_test

GEOMETRY, a FORTRAN90 code which contains many geometrical algorithms.

GEOMPACK, a FORTRAN90 code which contain routines for Voronoi diagrams and Delaunay triangulations.
