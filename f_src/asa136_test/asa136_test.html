asa136_test

asa136_test, a FORTRAN90 program which calls asa136() to divide M points in N dimensions into K clusters so that the within-clusters sum of squares is minimized.

asa136, a FORTRAN90 library which implements a clustering algorithm, by Hartigan and Wong. This is a version of Applied Statistics Algorithm 136.
