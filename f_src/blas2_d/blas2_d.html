BLAS2_D
Double Precision Real Matrix-Vector Basic Linear Algebra Subprograms

BLAS2_D, a FORTRAN90 code which constitutes the Level 2 Basic Linear Algebra Subprograms (BLAS), for matrix-vector operations using double precision real arithmetic.

The BLAS are a small core code of linear algebra utilities, which can be highly optimized for various architectures.
