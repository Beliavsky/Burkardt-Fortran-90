satisfy_mpi_test

satisfy_mpi_test, a FORTRAN90 code which calls satisfy_mpi(), which demonstrates, for a particular circuit, an exhaustive search for solutions of the circuit satisfy problem. This version of the program uses MPI to carry out the solution in parallel.
