ccs_io_test

ccs_io_test, a FORTRAN90 program which calls ccs_io() to read and write Compressed Column Storage (CCS) sparse matrix files.

Related Data and Programs:

ccs_io, a FORTRAN90 library which reads and writes sparse linear systems stored in the Compressed Column Storage (CCS) format.

Source Code:

  • ccs_io_test.f90 a sample calling program.
  • ccs_io_test.sh runs all the tests.
  • ccs_io_test.txt, the output file.

Test #1 and #2 write and read back the information for the "simple" 5x5 matrix with 12 nonzero entries. There are files associated with this.

  • simple_acc.txt, the values.
  • simple_ccc.txt, the (0-based) compressed column indices.
  • simple_icc.txt, the (0-based) row indices.
