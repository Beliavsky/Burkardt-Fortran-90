alpert_rule_test

alpert_rule_test, a FORTRAN90 program which calls alpert_rule() to define Alpert quadrature rules of a number of orders of accuracy for functions that are regular, log singular, or power singular.

Related Data and Programs:

alpert_rule, a FORTRAN90 library which can set up an Alpert quadrature rule for functions which are regular, log(x) singular, or 1/sqrt(x) singular.
