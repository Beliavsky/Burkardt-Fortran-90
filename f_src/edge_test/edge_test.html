edge_test

edge_test, a FORTRAN90 code which calls edge(), which defines some test functions in 1D, 2D and 3D for the detection of edges, and uses GNUPLOT to display images of the functions.

Related Data and Programs:

edge, a FORTRAN90 code which defines some test functions in 1D, 2D and 3D for the detection of edges or discontinuities. It uses gnuplot to display the jumps in curves and surfaces.
