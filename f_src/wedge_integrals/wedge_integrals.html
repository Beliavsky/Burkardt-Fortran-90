WEDGE_INTEGRALS
Integrals Inside the Unit Wedge in 3D

WEDGE_INTEGRALS, a FORTRAN90 code which returns the exact value of the integral of any monomial x^i y^j z^k over the interior of the unit wedge in 3D.

The interior of the unit wedge in 3D is defined by the constraints: 

        0 <= X
        0 <= Y
             X + Y <= 1
       -1 <= Z <= +1
