F90_FILES
Reading and Writing Files in FORTRAN90

F90_FILES, F90 codes which illustrate the use of files.

Languages:

A version of F90_FILES is available in a C version and a C++ version and a FORTRAN90 version.

Related Data and codes:

F90, FORTRAN90 codes which illustrate some of the features of the FORTRAN90 codeming language.

f90_stop_test, FORTRAN90 codes which illustrate how a FORTRAN90 code can print a text message or return a numeric code status value to the calling environment when executing the STOP statement.

FILUM, a FORTRAN90 code which can work with information in text files.

TABLE_IO, a FORTRAN90 code which reads and writes table files.

Examples and Tests:

DEFAULT_NAME examines what happens when you open a file anonymously, that is, without specifying a name, only a unit number.

  • default_name.f90, a code that opens several anonymous files.
  • default_name.txt, the output file;
  • fort.8, an anonymous file opened on unit 8.
  • fort.11, an anonymous file opened on unit 11.
  • fort.34, an anonymous file opened on unit 34.
  • fort.99, an anonymous file opened on unit 99.
