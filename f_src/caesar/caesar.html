CAESAR
Caesar Shift Ciphers

CAESAR, a FORTRAN90 code which applies a Caesar shift cipher to a string of text.

Languages:

CAESAR is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Programs:

ATBASH, a FORTRAN90 code which applies the Atbash substitution cipher to a string of text.

caesar_test

CHRPAK, a FORTRAN90 code which works with characters and strings.

FILUM, a FORTRAN90 code which can work with information in text files.

ROT13, a FORTRAN90 code which can encipher a string using the ROT13 cipher for letters, and the ROT5 cipher for digits.
