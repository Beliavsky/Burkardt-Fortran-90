tet_mesh_l2q_test

tet_mesh_l2q_test, a FORTRAN90 code which calls tet_mesh_l2q(), which converts a 4-node tetrahedral mesh ("tet mesh") into a 10-node tet mesh.

Related Programs:

tet_mesh_l2q, a FORTRAN90 code which reads information about a 4-node (linear) tet mesh and creates data defining a corresponding 10-node (quadratic) tet mesh;

Source Code:

  • tet_mesh_l2q_test.sh, runs all the tests.
  • tet_mesh_l2q_test.txt, the output file.

You may copy data defining an initial 4-node tet mesh:

  • mesh_nodes.txt, the nodes for the 4-node tet mesh.
  • mesh_elements.txt, the elements for the 4-node tet mesh.
  • mesh_l2q_nodes.txt, the nodes for the 10-node tet mesh.
  • mesh_l2q_elements.txt, the elements for the 10-node tet mesh.
  • tet_mesh_l2q.txt, the output file.
