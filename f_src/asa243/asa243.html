ASA243
Cumulative Density Function (CDF) of the noncentral T Distribution

ASA243, a FORTRAN90 code which computes the cumulative density function (CDF) of the noncentral Student's T probability density function, by Russell Lenth.

ASA243 is Applied Statistics Algorithm 243.
