LUHN
Luhn Check Digit for Error Detection

LUHN, a FORTRAN90 code which demonstrates the calculation and use of the Luhn check digit for error detection.

Licensing:

Languages:

LUHN is available in a C version and a C++ version and a FORTRAN90 version and a MATLAB version and a Python version.

Related Data and Programs:

ISBN, a FORTRAN90 code which can determine the check digit for an International Standard Book Number or report whether a given ISBN is valid.

luhn_test

UPC, a FORTRAN90 code which can determine the check digit for a Uniform Product Code (UPC) or report whether a given UPC is valid.

Source Code:

  • luhn.f90, the source code.
  • luhn.sh, compiles the source code.
