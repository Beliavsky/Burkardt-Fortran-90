allocatable_array_test
Allocatable Arrays as Output Arguments

allocatable_array_test, a FORTRAN90 program which demonstrates how a main program can declare an allocatable array, which is then passed to a subroutine which allocates it, initializes it, and returns it to the calling program for further use.
