triangulation_l2q_test

triangulation_l2q_test, a FORTRAN90 code which calls triangulation_l2q(), which reads information describing a triangulation of a set of points using 3-node ("linear") triangles, and creates a 6-node ("quadratic") triangulation.

Related Data and Programs:

triangulation_l2q, a FORTRAN90 code which reads information about a 3-node (linear) triangulation and creates data defining a corresponding 6-node (quadratic) triangulation;

Source Code:

  • triangulation_l2q_test.sh, runs all the tests.
  • triangulation_l2q_test.txt, the output file.

EXAMPLE is an example data set.

  • example_nodes.txt, the nodes for the 3-node triangulation.
  • example_elements.txt, the elements for the 3-node triangulation.
  • example.png, a PNG image of the 3-node triangulation, created by TRIANGULATION_PLOT.
  • example_l2q_nodes.txt, the nodes for the 6-node triangulation.
  • example_l2q_elements.txt, the elements for the 6-node triangulation.
  • example_l2q.png, a PNG image of the 6-node triangulation, created by TRIANGULATION_PLOT.
