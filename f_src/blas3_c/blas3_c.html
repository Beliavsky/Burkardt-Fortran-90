BLAS3_C
Level 3 Basic Linear Algebra Subprograms

BLAS3_C, a FORTRAN90 code which constitutes the Level 3 Basic Linear Algebra Subprograms (BLAS), for matrix-matrix operations using single precision complex arithmetic.

The BLAS are a small core code of linear algebra utilities, which can be highly optimized for various architectures. Software that relies on the BLAS is thus highly portable, and will typically run very efficiently.
