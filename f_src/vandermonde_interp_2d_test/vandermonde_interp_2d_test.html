vandermonde_interp_2d_test

vandermonde_interp_2d_test, a FORTRAN90 code which calls vandermonde_interp_2d(), which finds P(X,Y), a polynomial interpolant to data Z(X,Y) which depends on two independent variables X and Y, by setting up and solving a linear system involving the Vandermonde matrix.
