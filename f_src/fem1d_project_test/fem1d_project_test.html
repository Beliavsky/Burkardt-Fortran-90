fem1d_project_test

fem1d_project_test, a FORTRAN90 program which calls fem1d_project(), which computes the projection of data onto a given 1D finite element mesh.

Related Data and Programs:

fem1d_project, a FORTRAN90 code which projects a function into a function space associated with the finite element method (FEM), using least squares or projection methods, especially for cases in which data must be transferred from one mesh to another.

Source Code:

  • fem1d_project_test.sh, runs all the tests.
  • fem1d_project_test.txt, the output file.

POWERS starts with sample data for the vector function f(x)=[ 1, x, x^2], on a grid of 11 equally spaced nodes from 0 to 10, and projects this onto a piecewise linear grid of 6 equally spaced nodes from 0 to 10.

  • sample_powers_nodes.txt, the sample nodes.
  • sample_powers_values.txt, the sample nodal values.
  • fem_powers_nodes.txt, the FEM nodes.
  • fem_powers_elements.txt, the FEM elements.
  • fem_powers_values.txt, the projected FEM nodal values.
