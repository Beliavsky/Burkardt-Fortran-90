BANK
Bank Checksum Check Digit

BANK, a FORTRAN90 code which can compute the check digit associated with a US Bank Routing Number check digit, or it can report whether a 9-digit code is actually valid.

Related Data and Programs:

bank_test

ISBN, a FORTRAN90 code which can determine the check digit for an International Standard Book Number or report whether a given ISBN is valid.

LUHN, a FORTRAN90 program which computes the Luhn check digit for a string, and can validate a string.

UPC, a FORTRAN90 code which can determine the check digit for a Uniform Product Code (UPC) or report whether a given UPC is valid.
