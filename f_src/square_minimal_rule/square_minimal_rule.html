SQUARE_MINIMAL_RULE
Quadrature Rules for the Symmetric Square.

SQUARE_MINIMAL_RULE, a FORTRAN90 code which returns "almost minimal" quadrature rules, with exactness up to total degree 55, over the interior of the symmetric unit square in 2D, by Mattia Festa and Alvise Sommariva.
