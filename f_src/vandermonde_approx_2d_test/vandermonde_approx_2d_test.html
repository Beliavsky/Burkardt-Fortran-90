vandermonde_approx_2d_test

vandermonde_approx_2d_test, a FORTRAN90 code which calls vandermonde_approx_2d(), which finds P(X,Y), a polynomial approximant to Z which depends on two independent variables X and Y, by setting up and solving an overdetermined linear system involving the Vandermonde matrix.
