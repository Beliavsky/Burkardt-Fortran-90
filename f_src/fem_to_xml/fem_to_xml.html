FEM_TO_XML
Convert a Mesh from FEM to DOLFIN XML Format

FEM_TO_XML, a FORTRAN90 code which reads FEM files of node coordinates and element connectivity, which define a mesh of a 1D, 2D, or 3D region and creates a corresponding DOLFIN XML file.

These mesh files can be used as input to DOLFIN or FENICS.

Usage:

fem_to_xml prefix
where prefix is the common filename prefix:
  • 'prefix'_nodes.txt contains the node coordinates,
  • 'prefix'_elements.txt contains the element definitions.
  • 'prefix'.xml will contain the DOLFIN XML mesh file.
